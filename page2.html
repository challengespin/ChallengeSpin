ChallengeSpin

Règles du jeu

1. Ajouter les participants :

Chaque joueur doit entrer son nom dans le champ "Nom du participant" et cliquer sur "Ajouter". Les noms des participants seront affichés dans la liste des participants.

2. Définir les paramètres :

Déterminez ensemble le nombre maximum d'éléments (défis, anecdotes ou questions) que chaque joueur peut proposer. Entrez ce nombre dans le champ "Nombre maximum d'éléments" et cliquez sur "Définir".

3. Ajouter des éléments :

Chaque joueur, à tour de rôle et sans que les autres regardent, doit écrire un défi, une anecdote sur quelqu'un, ou une question. Sélectionnez le type d'élément (Défi, Anecdote ou Question) dans le menu déroulant. Entrez le contenu de l'élément dans le champ "Entrez un défi, une anecdote ou une question". Sélectionnez le participant à qui s'adresse l'élément dans le menu déroulant. Cliquez sur "Ajouter" pour enregistrer l'élément.

4. Tirer au sort :

Lorsque tous les éléments ont été ajoutés, cliquez sur "Tirer au sort" pour passer à la page de tirage au sort. Sur la page de tirage au sort, cliquez sur "Tirer un élément" pour sélectionner un élément au hasard. L'élément tiré sera affiché, indiquant le type (Défi, Anecdote ou Question), le contenu et le participant concerné. Répétez cette opération jusqu'à ce que tous les éléments aient été tirés au sort.

5. Réinitialiser le jeu :

Si vous souhaitez recommencer le jeu avec les mêmes participants ou en ajouter de nouveaux, cliquez sur "Réinitialiser". Cela effacera tous les éléments et remettra les paramètres à zéro.
